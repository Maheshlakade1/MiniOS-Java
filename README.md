# MiniOS – Command Line OS Simulator

## Description
MiniOS is a Java-based command line OS simulator that mimics basic OS commands for creating, deleting, listing, and moving files/folders inside a sandboxed directory.

It’s designed as a core Java project to demonstrate OOP, file handling concepts.

---

## Features
- List files/folders in the root directory (`list`)  
- Create files/folders (`create`)  
- Delete files/folders recursively (`delete`)  
- Move files/folders (`move`)  

---

## Technologies Used
- Core Java (Java 8 or higher)  
- Object-Oriented Programming (OOP)  
- File Handling
